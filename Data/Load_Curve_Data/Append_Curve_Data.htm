Append Curve Data

Append curves from successive mud log, MWD or LWD data files to existing curves within the ODF to create a continuous set of curves.

To append, the following conditions must be applied.

  • Curve mnemonics for the new curve(s) to be loaded, must be the same as for the existing curve(s).

  • The data sets must be sequential.

 

Refer to the Load Curve Data instructions below.  When selecting the Data Source Type, the following selections will automatically append the curve data; 

  • MWD/LWD

  • Mudlog

  • Mineralogy

In step 2 of the data import wizard, select Append Overlapping Curves and Append Non-overlapping Curves as required.

  1. On the GEO tab, select Import.

  2. In the Open dialog box, the Files of Type box lists the curve data types accepted. Select the type required.

  3. Locate and select the curve data file, then click Open. Step 1 of the Data Import Wizard will invoke.

  4. The curve data Type will be automatically detected.

  5. Select the Data Source Type. The selection will determine the default settings.

    1. Wireline    Identical mnemonics will NOT append automatically to preserve individual runs.

    2. MWD/LWD    MWD or LWD data curves may however be treated either as discrete run for each curve, or as continuous curves.

    3. Mudlog    Continuous data WILL automatically append.

    4. Mineralogy    Data WILL append regardless if the data continues or appears after a large interval.

    5. Other    Undefined source type. Manual append required.

  6. Select Next. Step 2 of the Data Import Wizard will open.

  7. Enter the data range to be loaded in the From and To boxes.

  8. Specify the Null (or pen-up) data values in the Use as Null data Value. The default is -999.25. If the data vendor has used a different value e.g. -999.99 define it here.

  9. By default, Treat Null data value as “empty” will be cleared. In which case, GEO will ‘pen-up’ when it encounters null values within a curve. The curve will be discontinuous over any null intervals. Select Treat Null data value as “empty” to draw a continuous curve. When a null value is encountered the curve will connect the previous valid value with the next valid value.

  10. Select or deselect Append Overlapping Curves and Append Non-overlapping Curves as required. Click for more information on appending curve data.

  11. Edit the curve Name, Mnemonic, Synonym, Units and Type as required.

  12. Elect to not load certain data by clicking on the column header. Or use the Select All or Deselect All buttons. Selected data will be black and deselected data will be red. If a mnemonic is not formatted in the curve default ini file, it will be blue.

  13. Click Finish to complete the wizard.
