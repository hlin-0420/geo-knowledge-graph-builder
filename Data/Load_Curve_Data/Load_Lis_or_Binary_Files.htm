Load Lis or Binary Files

Curve data in any of the binary formats (LIS, TIF, TAP or NTI) must be converted to LAS format prior to loading to GEO. You can use the Lis to Las Converter to convert the binary files into LAS file, although in practice it is not generally easier just to ask the vendor to provide the LAS data format instead. 

 

Once the files are converted to LAS format, they can be loaded the same way as any other ASCII data file.
