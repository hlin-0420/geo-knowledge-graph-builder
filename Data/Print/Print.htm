Print a Log

  1. Open the layout you want to print.

  2. On the GEO tab, select Print. The Print dialog box will open. 

  3. Select the printer or plotter Name, or save to a file by checking Print to file.

  4. Define the printer set up by selecting Properties. Click for further information on the printer set up.

  5. Print All of the log or specify a depth range to print by selecting Depth and entering the from and to depths.

  6. Enter the Number of copies required. 

  7. Choose Fit Plot to Page Width - the plot width will adjust to the paper size - or Fit Plot to One Page - the plot will adjust to fit onto one paper page. 
    Note. Fitting the plot to page width or one page will result in the sizing of the plot to be adjusted, altering the scale. It is recommended that print zooms are used to overcome this.

  8. Choose to Print Header, Print Zones, Print Trailer and Print Zone Reports by selecting/unselecting the boxes as required.  

  9. Define the Left and Top Margin.

  10. Click OK.
