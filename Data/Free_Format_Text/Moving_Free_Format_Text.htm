Move Free Format Text

Select the text and drag it to its new position.

Or, 

On the Home tab, select the Arrows Keys Nudge. Use the keyboard arrow keys to move the text to its new position.

 

Move free format text with precision

  1. Right click on the text and select Text Properties.
    The Free Format Text Entry sidebar will open.

  2. In the Location node, enter the new depth in the At box.

Note. Free format text can only be created on curve data tracks. But it can be relocated to display over any other track (Depth, Lithology, % Lithology or Text).
