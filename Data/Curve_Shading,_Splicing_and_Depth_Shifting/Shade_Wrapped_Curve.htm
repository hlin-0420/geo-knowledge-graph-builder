Shade Wrapped Curves

Shade between the wrapped (or out-of scale value) curves and the left or right track edge.  


For example, an ROP curve that has data range between 0 to 380 ft/min needs to be displayed with left and right edge scale at 0 to 300 ft/min. To indicate a fast drilling region, a red curve shading needs to be applied between left track edge and any ROP curve whose value is greater than 300 ft/min.


Ensure that the ROP curve has been wrapped right at least once, so the out-of-scale curve will be displayed on the track. 

  1. On the Curves tab, select Curve Shading. The Curve Shading Information dialog box will open. 

  2. From Curve Shading, select an unused number.

  3. Tick Display Curve Shade on Layout.

  4. Enter a curve shading Name.

  5. Select the Track where the ROP curve displays.

  6. In Shade From, select <left edge of track>.

  7. In To, select the wrapped ROP curve and tick Wrapped below. 

  8. Click Append and select the shading color as Red. 

  9. Select Update. 

 

Only the wrapped ROP (i.e. its value exceeding 300 ft/min) will then be shaded red.
