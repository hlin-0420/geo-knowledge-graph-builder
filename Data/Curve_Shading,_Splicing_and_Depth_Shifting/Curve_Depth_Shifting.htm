Curve Depth Shifting

The depth of curves can be shifted to shallower or deeper depths using the depth shifts feature. To access this, on the Depths tab, select Curves and Global Depth Shifts. The Depth Shift Specifications dialog box will open. From here you can perform the following actions.

 

Global Depth Shift All Curves

Shift all curves by a particular amount.

  1. In the Depth Shift Specifications dialog box, enter the Shift value, that you wish to move all the curves by.

  2. Check Apply Global Shift to Curves, then Apply. 
    Note. It is not possible to shift a section of curve when performing a global depth shift. Only the whole curve can be shifted.

 

 

Depth Shift Individual Curves

Shift the depths of individual curves. This is also knows as rubber banding. 

  1. In Depth Shift Specifications, select the individual curve you wish to shift from Curve Related Depth Shift.

  2. Click Create Curve Depth Shift – two depth shift rows will appear in the grid

    1. The Depth column, states the first (start) depth and second (end) depth of the selected curve. The values can be changed by typing in the cell. 

    2. The Shift column, states the amount the depth point has been shifted by.

  3. To shift the curve, enter the amount you wish to shift the first depth and second depth by, in their corresponding Shift cell.

  4. Add more depth points to shift by selecting Insert Row.

  5. To delete inserted rows, select them and click Delete Row. 

  6. Click Exit. 

 

 

Apply Same Rubber Banding Shifts To Other Curves
You can copy the depth shift pattern specified for an individual curve, to another curve. 

  1. In Depth Shift Specifications, select the curve with the shift pattern you wish to copy in Curve Related Depth Shift.

  2. Click Copy Curve Depth Shift.

  3. Select the curve you wish to apply the shift pattern to in the Curve Related Depth Shift drop list.

  4. Click Paste from Curve x (the x will be the number of the curve), then Exit. 

 

 

Remove the Depth Shift

  1. In Depth Shift Specifications, select the curve you wish to remove the depth shift for in the Curve Related Depth Shift drop list. 

  2. Select Remove Curve Depth Shift. GEO Message:816 will open.

  3. Click Yes, then Exit.

All Depth Shifts for the curve will be deleted and the original (unshifted) depths restored.

 

 

View Depth Shift Details
There is an additional dialog box accessed via the Depth Shift Specifications where you can view each shift depth point and the data value associated with it. This is called the Curve Depth Details. 
In this dialog box you can also remove the shift and revert to the original data, or you can preserve the new shifted data with no evidence of a shift.

  1. In Depth Shift Specifications, select Curve Depth Details. The Curve Depth Details dialog box will open. 

  2. Select a Curve you wish to analyse.

  3. The Current Depth, Computed Depth and Value or each data point in the curve will be listed.

  4. To remove the depth shift, select Revert to Original Depths, then Remove. GEO Message:816 will appear - click Yes.

  5. To make the shifted depths the 'true' data points of the curve (there will be no evidence this curve was depth shifted), select Preserve Current Depths, then Remove. GEO Message:826 will appear - click Yes. 

  6. Click Exit to return to the Depth Shift Specifications. The curve selection will need to be refreshed to reflect any changes made in the the Curve Depth Details.
