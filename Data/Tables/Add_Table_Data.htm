Add Table Data

Tables store tabular information, that relates to specific symbols on the plot.  

This information may appear in the plot header or trailer, with its corresponding symbol on the log.

For example, shot number information recorded in the SWC table, could be displayed next to the SWC symbols in the plot.


Add table data

There are two ways to launch the table definition dialog box in which you can enter data.

  • If there are tables in the header, double click on it to launch the Table Definition dialog box. 
    Or,

  • If there are no header tables, on the Home tab, select Create Table from Template. 

  • Select a Category and Select a Table Template.

  • Click Apply. The Table Definition dialog box will open.

 

  1. Select the cell to be edited and enter the data.
    Note. The Start Depth defines the position of any symbols associated with the data.

  2. Use the tab key to move from one cell to another.

  3. At the last cell, you will need to hit Enter on your keyboard to confirm the data.

  4. Use the Append Row, Insert Row and Delete Row to edit rows.

  5. Click Exit.
