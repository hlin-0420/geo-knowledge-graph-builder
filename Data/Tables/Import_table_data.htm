Import table data

  1. In another application, such as MS ExcelTM, select or highlight the data cells that you want to import, using the following rules.

    1. The first selected column must be depth values.

    2. The subsequent columns should be in the same order as their corresponding columns in the table.

    3. The first selected row must be the values (cannot be the remarks or column titles).

  2. Copy the data to clipboard.

  3. In the GEO Home tab, select the table from the Select Definition box. 
    If the table has not been created, select Create Table from Template then pick a Category and Select a Table Template.

  4. Click Clipboard in the Load Rows From section. This will import all the information from the clipboard into the selected table.

  5. Click Exit.


Append table data

Once you have imported or added table data into any of the tables, it is possible to import further sections of data from outside of GEO.

  1. In the raw data file, for example Microsoft ExcelTM, highlight the new data.

  2. Ensure the columns are in sequence with that of the table in GEO and that the depth values are appropriate.

  3. Copy the data to clipboard.

  4. In the GEO Home tab, select the table from the Select Definition box.

  5. Click Clipboard from the Load Rows from section. A confirmation dialog box will open. Select,

    1. Yes to append the rows

    2. No to replace existing rows

    3. Cancel to abort the import

  6. Click Exit.
