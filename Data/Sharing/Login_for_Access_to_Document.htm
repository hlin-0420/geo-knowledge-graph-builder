Login for Access to Document

In order to preserve secrecy on any work that you do within GEO, it is possible for you to set password protection on your files. There are three levels of protection you can apply to a document: Update, View / Print and Export.

 

When attempting to access a document that is password protected you will be faced with a drop down option box and a text box.

  1. Select File, then Open/Import to locate the desired document.

  2. Click on the small arrow at the end of the drop down option box.

  3. Select Update, View / Print or Export.

  4. In the box below type in the password for the appropriate selection.

  5. Click OK

 

Note. Clicking Cancel will result in the appearance of a warning message and termination of the GEO program.

 

Once access is granted you may perform the relevant modifications or actions dependent on the access level you have reached. Any option that is ‘grayed’ out will not be available on the current level of access.

 

Note. Passwords are not case sensitive.
