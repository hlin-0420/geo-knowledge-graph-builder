Export GEO Plot to TIFF

The TIFF export option can be used to replicate the GEO plot in to a commonly used graphics format. This file can then be viewed within many third party graphics applications.

 

The TIFF feature can be used to export the plot as a whole, however for large plots, in both width and length there are some associated limits to be aware of.

 

There are two controls for exporting to TIFF format - DPI and Page Height. Both of these options can be edited on the Edit tab in  Edit GEO Settings ( General tab).

 

 

  • DPI
    DPI or Dots per Inch, relates to the resolution and quality of the output. The higher the DPI the better quality the image will be. When an ODF is viewed on screen at 100% please bear in mind that this will be approximately 96dpi, thus increasing the DPI for TIFF export will also serve to increase the respective size of the image by that factor.
  • Page Height
    The page height relates to the maximum length in pixels of your TIFF page.
    If your log exceeds 2048 pixels wide then your max page height will be 32000 pixels
    If your log is less than 2048 pixels wide then your max page height will be 64000 pixels
    Therefore if your log is longer than 64/32k pixels then you will output a multi-page TIFF file.

 

Export to TIFF

  1. On the Depth tab, enter the Index Range you wish to export in From and To. 
  2. On the Export tab, select Export As... then TIFF Tagged Image Format. The Export To Tiff File As dialog box will open. 
  3. Browse to save location and enter a file name.
  4. Click Save.

The following video selects WMF as the export format. Simply replace this selection with TIFF.

 

Tip. If you experience memory issues during this process, try reducing the DPI.
