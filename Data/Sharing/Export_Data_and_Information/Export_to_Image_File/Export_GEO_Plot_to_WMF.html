Export GEO Plot to WMF

In WindowsÔ, most major applications support a format called Windows Metafile or WMF.

WMF is useful for transferring a small section of the plot into another application such as MS WordÔ or MS PowerPointÔ.

 

We do specify a small section of the plot, primarily because of two reasons;

  • WMF format is not geared to cope with the length of the plot that GEO usually handles.
  • Many applications may run out of memory when handling very complex pictures as created by GEO.

 

GEO can also export the plot to a Placeable WMF.

 

The only difference between the standard WMF and the placeable WMF is the presence of an additional 20-byte header in the latter.

 

Export a section of plot to WMF

  1. On the Depth tab, enter the Index Range you wish to export in From and To. 

  2. On the Export tab, select Export as... then either WMF-Standard Metafile, Placeable WMF or WMF-Metafile picture to clipboard.

  3. An Export To dialog box will open (unless you select WMF-Metafile picture to clipboard).  Enter a File name and the directory where it is going to be stored.

  4. Click Save.

If the plot is exported as WMF-Metafile picture to clipboard, bring it to the other application by pasting it.

 

Tip. The best route to bring the plot to your MS PowerPointÔ for presentation is to specify the zone of interest depth range and export the plot either as WMF-Metafile picture to clipboard or EMF-Enhanced Metafile to clipboard. Then paste it to MS PowerPointÔ .
