Export GEO Plot to CGM

The CGM (Computer Graphic Metafile) format can be used to transfer a plot to a large plotter (e.g. HP DesignJetÔ) through CGM interpreter software (such as Zeh GraphicsÔ or SDIÔ software).

 

One reason for doing so is to circumvent the length limitation (109 inches) of certain printer-drivers operating under WindowsÔ. This limitation does not of course arise when plotting under the UNIX platform.

 

GEO can export to CGM files with either 16 or 32-bit precision.

The 16-bit CGM complies with ISO-8362 binary specifications and may not be able to cope with very long plots.

The 32-bit CGM, on the other hand, is equipped to handle any length of plot and is compatible with PIP (Petroleum Information Profile).

 

Export Plot to CGM

  1. On the Edit tab, select Headers and Trailers. The Plot Header and Trailer Specifications dialog box will open. 
    Tip. If a trailer component such as a spreadsheet was loaded with the OLE object check box selected, you can remove the OLE object with the following simple procedure.

    1. Double-click that spreadsheet in the trailer in GEO to open it in the native application (perhaps MS ExcelÔ).

    2. Copy it to the clipboard and load it back at the Plot Header and Trailer Specifications dialog box, with the same name, but this time ensure the OLE object check box is clear (no tick).

  2. On the GEO menu, select Print and select a printer that will allow you to set the paper size at a certain value, so the paper’s width is greater than the plot width. Save these settings but do not print. 
    Note. The exported plot will be cut off on width, if the paper size set at this Print Setup is less than the original plot’s width.
    Tip. Our experience dictates that the Epson Stylus Color 1520Ô printer driver is a good driver for creating reliable CGM exports. Note that although this driver may be the optimum to use for CGM production, it is not necessary to physically have such a printer available or attached to your computer; only the driver need be installed (from https://www.epson.co.uk/support?productID=3838&os=22#drivers_and_manuals) and the printer selected within GEO.

  3. On the Edit tab, select Edit GEO Settings. The Registry Editor dialog box will open.

  4. Open the CGM tab and ensure that the fonts are correctly mapped between Windows and CGM Interpreter (such as ZEH GraphicsÔ or SDIÔ software); for this you may need assistance from the personnel within your IT Department or Help Desk. Select Apply. 

  5. Open the CGM Advanced tab and check the settings.
    Note. The defaults are usually good in most cases.

  6. On the Export tab, select Export as... then CGM. The Export to CGM (32 bit) will open.

  7. Enter the File name and specify the path to save to.

  8. Click Save.

 

For output to the plotter, such as the HP DesignJetÔ series transfer the corresponding CGM format and its parameter file (with par extension) to the plotter by using FTP (File Transfer Protocol) to the UNIX platform.
