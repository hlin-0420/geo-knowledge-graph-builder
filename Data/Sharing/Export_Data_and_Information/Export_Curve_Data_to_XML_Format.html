Export Curve Data to XML Format

XML (eXtensible Markup Language) is a platform-independent meta-markup language that provides a format for describing structured data.

 

XML provides a structural representation of data that has proved broadly implementable and easy to deploy across all platforms.

 

The specifications of XML have been laid down by the World Wide Web Consortium (W3C, see www.w3c.org for details), an international consortium founded in 1994 to develop standards for the Web.

 

WellLogML is an XML standard adopted by the Petrotechnical Open Software Corporation (POSC) to exchange well log data (see www.posc.org/ebiz/WellLogML). SDC Geologix is a full member of POSC and collaborates in the development of these standards.

 

WellLogML is a system for encoding well information for the Web that is flexible and extensible, suitable for interaction with external software on any platform.

 

WellLogML is similar to LAS, in that it specifies how well log data is organized in terms of Well Information, Curve Information, Parameter Information, etc.

 

Export Curve Data as XML

  1. On the Export tab, select Export Curves. The Export Curve Data dialog box will open. 

  2. In Export Format, select XML.

  3. Select one or more curves to export
    Note. These curves MUST have the same depth interval.

  4. Change the Start Depth, End Depth and Null Value if necessary.

  5. Choose the appropriate value for the Parameter Information from and the Well Information from.

  6. Click Export, choose a folder and file-name to export to and click Save.
