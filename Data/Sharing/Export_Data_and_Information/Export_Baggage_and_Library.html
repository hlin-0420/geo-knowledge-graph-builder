Export Baggage and Library

You can export the complete set of library and baggage components from within an ODF or ODT. This can be used for the sharing of your library, or for checking the content of an ODT when in construction.

 

  1. On the Tools tab, select Export Library. The Browse For Folder dialog box will open. 

  2. Select a location to export to.

  3. Select Make New Folder and enter a suitable name. 

  4. Click OK.

 

Tip. To input library files into another ODF/ODT, use the multi-file load feature - on the Tools tab, select Library Tools (this opens the Library Manager) and select the relevant category radio button. I.e. Symbols, Lithology etc. Browse to the file, select it and click Open.
