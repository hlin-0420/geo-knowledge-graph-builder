Send OUF as Mail Attachment

In most cases, the OUF will be sent from one place to another as an e-mail attachment. There are two ways to place the OUF as an e-mail attachment:

  • By attaching the OUF files manually within your e-mail software itself.

  • Using Send OUF Updates via Mail (from the GEO menu) feature in GEO.

 

If the Send OUF Updates via Mail feature is utilized, you do not need to click the Save OUF Updates to save the sequential OUF. Pointing at Send OUF Updates via Mail and clicking the Updates option automatically saves the next sequential OUF file.

 

For first time usage, you need to go through the following procedure to set it up:

  1. On the GEO meny, select Send OUF Updates via Mail to open the Choose Profile dialog box. It allows you to select the information service by which the OUF file will be mailed. You can set up your choice of information service using the Windows Messaging Set-Up Wizard. Once selected, this invokes your installed e-mail software (for example MS Outlook) and opens the Message dialog box.

  2. The Message dialog box consists of two tabs - Message and Options.

    1. The Message tab allows you to enter the address (or addresses) of the OUF recipient(s). You can enter any desired text here, which is then mailed along with the OUF file.

    2. The Options tab allows you to set several options relating to the conditions under which the mail is sent. The Importance Level, Delivery Options and Tracking Options can all be set from this tab.

 

Next time you click Send OUF Updates via Mail from the GEO menu, the corresponding OUF will be saved and it will open your e-mail software with the OUF automatically created as the attachment. You can type the message as normal prior to sending this e-mail.

 

In the event that you not complete the Send OUF via Mail sequence and quit partially through the steps (e.g. you do not really send the e-mail), GEO will inform you that the OUF file has already been saved (specifying the file path) and that the particular OUF must be sent manually (as an e-mail attachment, for example).

 

If you click Send Range of OUF Updates via Mail, a Re-Save range of updates dialog box opens. Click the row which details the OUF sequence number to be saved over. Click Save and this will create an underscore OUF number that can be sent as an e-mail attachment.
