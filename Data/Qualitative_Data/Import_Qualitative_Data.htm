Import Qualitative Data

Qualitative data can be imported from .txt file.


The .txt file needs to be tab delimited and have a depth range that is within the global depth range of the receiving odf. 


The qualitative track receiving the .txt file data must be formatted to accept the data. The Logging Frequency, Differential Color On options and Graduation name should match that of the data file to be imported. Click for more details on setting up a qualitative track. 

  1. On the Edit menu, select Track Settings. The Track Settings dialog box will open. 

  2. Select the qualitative track and select Details. The Qualitative Track Information dialog box will open. 

  3. Select Import from File.

  4. In Look in browse to the .txt file, select it and Open. 

  5. Click OK.

  6. Click OK to exit the Qualitative Track Information.

  7. Click OK to exit Track Settings.
