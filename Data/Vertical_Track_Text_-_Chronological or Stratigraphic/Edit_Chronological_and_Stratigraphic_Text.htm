Edit Chronological or Stratigraphic Text

  1. Double click on the text you with to edit or right and select Track Text properties. The Track Text Entry will open.

  2. Edit the properties as required and click Apply then Close.

 

Note. Double clicking on an existing text entry will not invoke the Track Text Properties, but will add a new text entry.
