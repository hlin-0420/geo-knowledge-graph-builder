Add a Graph

The process of adding a Graph (a dynamic GGD file) to your ODF or ODT is much the same as adding anything to a header or trailer. 

 

  1. If you are using Layouts, select the one you wish the graph to appear on.

  2. On the Edit tab, select Headers and Trailers. The Plot Header and Trailer Specifications dialog box will open. 

  3. Click Load. The Open dialog box will open. 

  4. Locate and select the .ggd file you wish to import, then click Open. 

  5. The .ggd file will now appear in the Existing Specifications dialog box.

  6. Select the .ggd file and choose to Add to Header or Add to Trailer as required.

  7. Select the .ggd file in the appropriate Header or Trailer section and define the Position and Zoom, X and Y co-ordinates for the graph. 

  8. Click Exit to apply the changes.

Once a graph has been loaded into the header specifications it can be displayed on multiple layouts. Select the layout you wish to place the graph on and repeat steps 6 onward. 

 

Tip. To make instant changes to a graph, double click the graph.
