System Requirements

The minimum system requirements, 

 

Operating system

  • Windows 10

  • Windows 11

 

Processor

  • 1Ghz or higher. x86 or x64

 

Memory

  • 4GB of RAM (Windows 10, Windows 11)

 

Hard Disk

  • 700 MB (excluding Microsoft .Net Framework 4.8 installation)

  • 4.6 GB (including Microsoft .Net Framework 4.8 installation on x86/x64 processor)

 

Display

  • 1280 × 800 display

 

Other

  • A two-button mouse.

  • Recommended printers are those that can handle continuous media to enable printing of long logs supported by a Windows printer driver.

  • Dongle Port USB-A Compatible
