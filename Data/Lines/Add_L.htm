Add Line

Draw a line on the plot.

  1. On the Home tab, select Line. The cursor will display the text ST.LINE to indicate that the mouse may be used to define the starting position of a line.

  2. Place the mouse at the point where the line should start and click. The cursor text will change to EN.Line.

  3. Place the mouse at the point where the line should end and click. The Line Definition dialog box will open.

  4. Select the line Type and Color.

  5. Re-position the line by entering the depth and X-position of the start and end line, in Start and End point attributes.

  6. You can also change the anchor for the start and end of line, by selecting from the Anchor box.
    Note. The X-position depends on the anchor point. You need to enter to which track the anchor is to be made, if you select either Left or Right edge of track in the Anchor section.

  7. To add an arrow to the line, select from the Arrow Styles.

  8. Click New to enter free format text that will be associated/related to the line.
    The Free Format Text Entry sidebar will open. Click to learn how to add free format text.

  9. Click OK to draw the line on the log.

Note. During line copy-paste process, only a horizontal line can be pasted in the plot with the Start point independent from the mouse pointer position. If a tilted (incline) line is copied, it will be pasted with the Start point at the mouse pointer position (when you do a right click over the plot before clicking Paste).


Tips.

If the <Shift> key is depressed before the end point is defined, a vertical line will be drawn from the starting point to the depth where the end point is clicked.


If the <Ctrl> key is depressed before the end point is defined, a diagonal line will be drawn from the starting point to the end point.


Please note that this type of line can only be created with custom or solid lines. 


To terminate the line definition mode, prior to defining the end point, press the <Esc> key.
