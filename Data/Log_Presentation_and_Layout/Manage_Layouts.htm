Manage Layouts

The layout manager dialog box can be accessed by selecting Layout on the Tools tab.


It allows you to administer and define individual properties for individual layouts. 


The layout manager details information for all layouts in the current odf or odt file. 

 

Change Layout Name

  1. Select the layout in the Select Layout list.

  2. Edit the Name.

  3. Click Update.

 

Define Layout Order
The order that the layouts appear in the Layout list on the Home tab, can be changed.

  1. Select the layout in the Select Layout list. 

  2. Use the Layout Order down or up arrows to adjust it’s position.

  3. Click Update. 

 

Viewer Enable

A permission can be set so only certain layout(s) within a ODF can be view by either GEOView or GEOeView. 

To enable/disable a layout to be viewed.

  1. Select the layout in the Select Layout list.

  2. Check Viewer-enabled to allow the layout to be viewed in GEOView and GEOeView. To disable its ability to be viewed, clear Viewer-enabled.  

  3. Click Update. 

On the Select Layout list, an asterix will display beside layouts which are viewer enabled. 


When another user opens the ODF using GEOView or GEOeView, only the layouts which are viewer enabled will be visible in the Layouts list on the Home menu. 

 

Lock Index-Range

Lock the depth/time range of the selected layout. 


This is useful when sharing files with users that are viewing the log with GEOeView or GEOView. This feature is used to highlight specific areas, or to shield or protect sensitive information above or below certain depth ranges. 

  1. Select the layout in the Select Layout list. 

  2. Select Lock Index-Range.

  3. Click Update. 

On the Select Layout list, an L will display beside layouts where the index range is locked.

Assign GEO Report to Layout

A layout can be assigned as a ‘report layout’.  

  1. Select the layout in the Select Layout list. 

  2. From the Report list select DGR or End of Well Report.

  3. Define the Run Date Time of the report you wish to view on the layout

  4. Click Update.

The Date Range and Depth Range of the Run Date Time selected are detailed below the drop lists

 

Delete a Layout

  1. Select the layout in the Select Layout list. 

  2. Click Delete.

  3. Click Yes to the confirmation message.
