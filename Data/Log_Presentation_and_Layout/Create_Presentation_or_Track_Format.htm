Create Presentation or Track Format

Use the following procedure to create a new track format. 


This can be saved as a View file which can be imported into other ODF or ODT files. 

  1. Determine the width of the plot presentation (usually based on the paper size to print it) and the width of each track (either in inches or millimeters).

  2. Determine the track type for each track; Depth, Linear, Logarithmic, Lithology, % Lithology, Text and Qualitative. For each logarithmic type, establish the number of log cycles and what should be the start number (1 to 9) or the log cycle. 

  3. On the Edit tab, select Track Settings. The Track Settings dialog box will open. 

  4. On the Main tab, click number 0 and enter a Track Name.
    An example setup for a database (allowing toggling between Lithology Log and Composite Log layouts) might be:
    Track 0 as Depth track
    Track 3 as Lithology (interpreted) track
    Track 5 as Correlation (Linear) track (for ROP, GR, SP)
    Track 7 as Chromatograph Gas (Logarithmic) track
    Track 16 as Wireline Resistivity (Logarithmic) track (for ILM and ILD)
    Track 23 as Lithology Descriptions (Text) track. 
    Track 32 as Shows (Qualitative) track. 

  5. Select Display Track turn on its display.

  6. Enter the track Left Position and Right Position.

  7. Select the Track Type.

  8. The following step is dependent on the track type selected.

    Linear

    Enter the number of Vertical Grid Lines and Vert. Major Every.

    Lithology

    You have the option to define a grid value which restricts the depth of drawn lithology to multiples of that value. Select Snap to Grid and enter a Grid Value. 

    Qualitative

    Details will need to be defined. Click here for further information. 

    Logarithmic

    Enter the Log Cycles and Log Cycle Start. 

    Text

    Select the Track Sub-type;
    Remarks (for horizontal text entries) or,
    Chronological/Stratigraphic (for vertical text).

  9. Select Apply.

  10. Go to the next track number and repeat the steps 4-9 until all tracks are set up as required.

  11. On the Legends and Index tab, determine the legend display for each track. For more information, view the Adjust Track Legend video in the Useful Videos folder in Help.

  12. Click OK or Close to exit.
