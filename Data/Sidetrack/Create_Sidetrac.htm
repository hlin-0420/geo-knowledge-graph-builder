Create Sidetrack

A sidetrack well can be created from any ODF you have saved. This will allow you to select an existing well, and at a given depth truncate that well so that from that Kick Off Point you can add new data and new interpretations for the new sidetrack.

 

To create a Sidetrack:

  1. Ensure your ODF is saved (on the GEO tab, select Save)

  2. On the GEO tab, select Create Sidetrack... The Create Sidetrack dialog box will open. 

  3. Enter the Kickoff Depth for Sidetrack.

  4. Click OK. The Save Sidetrack to Document box will open. 

  5. A file name and save location for your sidetrack will be suggested. Change if required. 

  6. Click Save.

  7. The ‘sidetrack’ ODF will open on returning to GEO.
