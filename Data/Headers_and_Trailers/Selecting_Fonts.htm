Selecting Fonts

The font type, size and style of your vector object (VOB) font can be defined in the Fonts for Vector objects dialog box.

 

This dialog box opens when the user clicks Fonts or Choose Font from any dialog box that gives you the choice to select the fonts.

 

 

The Font section displays all possible font types. Only scaleable fonts are displayed for selection.

 

Note. The font type that is available is dependent on the capabilities of the currently selected printer driver. Only fonts that are available on both the currently selected printer and screen device will be displayed as choices.

 

If the plot is created selecting a printer that supports TrueType fonts, these fonts will be available as choices. When a currently selected font that is not supported by both the printer and the screen device is entered in this dialog box, the font field will change into spaces.

 

You can also choose the Font Style, Size, Color and to Strikeout and/or Underline the text.

 

Click OK to save any changes.
