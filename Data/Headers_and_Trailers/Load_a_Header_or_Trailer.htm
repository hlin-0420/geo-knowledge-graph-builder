Load a Header or Trailer

To add headers or trailers to your log, the required file should be present in the Existing Specifications of the Plot Header and Trailer

Specifications dialog box. 

 

If the files are not present, they need to be loaded into the ODF.

 

On the Edit tab, select Headers and Trailers. The Plot Header and Trailer Specifications dialog box will open. 

 

Load  a VOB or GGD header or trailer File

  1. Click Load. The Open dialog box will open. 

  2. Select the VOB or GGD file that you want to load and click Open. 

 

Load other types of header or trailer file (map, picture or spreadsheet)

  1. Copy the picture or file content to the clipboard.

  2. In Load Specification from Clipboard tick Picture and enter a file name in the text box below. 

  3. Click Load. 

 

Load Zones as a header or trailer file

Display zone reports based on global depth ranges or manual zones in the header. This gives information on a variety of curves,

lithology and computed curve parameters.

  1. Click Create From Query Definition. The Select Zone Definition dialog box will open.

  2. Select the zone query you wish to add to the header/trailer and click OK. 

 

Note. When adding the zone query from the Existing Specifications to the Header/Trailer Specifications the Zone Query Depth

Range dialog box will open. Here you can choose to display report data within the Global Depth Range, the Current GEO Report or

All Zones of Type (all data from the zone type selected).

 

The newly loaded file will be listed in Existing Specifications. From here it can be added to the header and trailers on your log.
