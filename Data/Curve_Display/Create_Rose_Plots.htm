Create Rose Plots

Plot Azimuthal distribution data as rose plot diagrams on the log plot.

 

  1. Load Azimuthal curve data into GEO

  2. On the Curves tab, select Rose Plot Definition. The Rose Plot Definition will open. 

  3. On the Definition tab, select an unused Rose Plot.

  4. Enter a Name.

  5. Select a Track to display the rose plot on.

  6. Select the Curve containing the azimuthal data.

  7. Define the Rose Plot Type - the frequency of azimuthal data occurrences - as a Count or Percentage.

  8. Enter the data range of azimuthal curve to be considered in the rose plots in the Depth From and To. The maximum and minimum depths of the azimuthal curve selected will display on the right.

  9. Define the Sample Data Range to be considered per rose plot.

  10. Define the Rose Plot Diameter.

  11. Align the rose plot to the Left, Center or Right of the track.

  12. Define the distance that the left edge of the rose plot displays from the left, center or right alignment (defined in the previous step) in the X Offset withing the Track.

  13. Define the Data Bar Color,

    1. Based on a Tadpole Definition -  select a tadpole definition from the drop box and click the Dip Type to define which are used in the Rose Plot by entering Yes/No in the Use for Rose Plot column. This option is only available if a tadpole definition it available in the log.
      or,
      Choose to display data as a Single colour (select the Color button to define).

    2. Check/clear Fill to fill/un-fill the data bar.

    3. Define the Pen Width of the data bars outline.

  14. The In-Use box will be checked if the current rose plot definition is being used. To delete the selected rose plot definition clear this check box and click Apply.

  15. To display the rose plot in the current layout check Display on Layout and click  >> to control the display on other layouts (hold CTRL to select/deselect more than one layout).

  16. To duplicate this information in another rose plot definition, select Create Copy and enter a different name (if required).

  17. To save the definition properties, click Apply.

 

Define the rose plot scale and display

  1. On the Curves tab, select Rose Plot Definition. The Rose Plot will open.

  2. Open the Scale & Display tab.

  3. The azimuthal data values are plotted around the periphery of the rose plot on the Angular Grid;

    1. Define the Start Value (displays at due north) and End Value.

    2. Define the Interval that the azimuthal data will be grouped into.

    3. Check/clear Display Grid to display/hide the angular grid.

    4. Define the Pen Width of the angular grid.

    5. Define the Color of the angular grid.

    6. Check/clear Display Scale Values to display/hide each angular grid value.

    7. Define the Font of the scale values.

    8. Check/clear Display Outer Angular Grid to display/hide the outer angular grid (the circumference outline) of the rose plot.

  4. The azimuthal data frequency is plotted on the Radial Grid;

    1. Define the Min Values, Max Values and Interval.

    2. Check/clear Display Radial Grid to display/hide the radial grid.

    3. Define the Pen Width of the radial grid.

    4. Define the Color of the radial grid.

    5. Check/clear Display Scale Values to display/hide each radial grid value.

    6. Define the Font of the scale values..

  5. Choose the Plot Background Color.

  6. Click Apply and Close.
