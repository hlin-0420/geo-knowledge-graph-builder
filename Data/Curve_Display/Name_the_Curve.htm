Name the Curve

Change the curve name, its mnemonic, as well as its measurement unit.


Use the Settings tab

  1. Select the curve on the plot. The selected curve will retain its color and all other curves will become grey. The Settings context tab will appear.

  2. In the Name section, change the curve Name, Mnemonic and Units.

  3. Hit Enter to implement changes. 



Use the Curve Settings sidebar

  1. On the Curves menu, select Curve Settings. The Curve Settings sidebar will open.

  2. Select the Curve, then Close the list. 

  3. Change the Name, Mnemonic and Units. 
    Note. When editing the mnemonic, remember that any successive curves with that mnemonic may no longer append to those already loaded, because the original curve mnemonic has been changed.

  4. Select Apply.
