Save the Curve Display Settings (Defaults)

The display settings for each curve, such as curve mnemonic, scale, pen type, pen color, name and measurement units, auto populate capabilities etc., can be saved as defaults for future usage. 

 

  1. On the Curves tab, select Curve Settings. The Curve Settings sidebar will open.

  2. Select the Curve, then Close the list. Ensure its display properties are as required. 

  3. Select Save as Defaults.

Do this for each curve you wish to save the default display settings for. 


The defaults will then be stored within the geocurve.ini file, which usually resides in the GEO library directory. 
The geocurve.ini file can be loaded into other ODF’s and the saved defaults applied to the curves within.
