Multiple Scales for Curve

You can have multiple scales for a curve in a plot. This is useful, for instance to change the curve scale at the middle of the plot, to have the curve always within the scale range (i.e. not wrapped).

 

  1. On the Curves tab, select Curve Settings. The Curve Settings dialog box will open. 

  2. Select the Curve, then Close the list. 

  3. On the Multiple Scales tab, enter the first scale by adding a Start Value and End Value.
    Note. These values will be same as Data value at Left and Right track Edge boxes on the Scales tab.

  4. Select Append Row to add another scale for another depth range. 

  5. Enter the depth this scale will start from in Depth From and its scale in the Start Value and End Value. 

  6. Select Apply to implement the changes.
