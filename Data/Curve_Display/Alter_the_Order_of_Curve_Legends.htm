Alter the Order of Curve Legends

The order of curve legends on a plot is determined by the curve order in curve settings list.  


By default, the order in the curve settings list is the same order as the order in the file, from which the curves were loaded.

 

  1. On the Curves tab, select Curve Settings. The Curve Settings sidebar will open.

  2. Open the Curve list and in Sort by, select User Defined.

  3. Select the curve and use the Order up and down arrows to arrange the curves.

  4. Click Close. 

  5. Click Apply. 


Note. The Curve ID (number) remains the same, even though the order changes. If you choose to Sort by Number, the curves appear in the same order as when loaded (numerical order).
