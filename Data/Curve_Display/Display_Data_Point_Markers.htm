Display Data Point Markers

Data points in a curve can be marked on the plot. 


However, since these appear at actual data points, using such markers on a curve with a sample frequency of 0.5 ft (6 inch) on a 1:500 scale would cause the markers to overlap each other.


Data point markers are therefore normally used on those data, which may not have a fixed sample frequency, such as shale density, or core porosity or permeability analyses.

 

  1. On the Curves tab, select Curve Settings. The Curve Settings sidebar will open.

  2. Select the Curve, then Close the list. 

  3. On the Data tab, open the Point Marker node. 

  4. Select the marker Style.

  5. Select the marker Color.

  6. Use the Size up and down arrows to increase or decrease the marker size. Or type the size. 

  7. Select the Highlight Color. This is the color the data points will appear when they are selected on the plot. 

  8. Specify the number of Display Markers in Curve Legend (i.e. if 2 was entered, 2 markers would display in the curve legend). 

  9. Select Apply to implement the changes.
