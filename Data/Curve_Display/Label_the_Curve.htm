Label the Curve

Display the curve mnemonic or name alongside the plotted curve. 

 

  1. On the Curves tab, select Curve Settings.  The Curve Settings sidebar will appear.

  2. Select the Curve, then Close the list. 

  3. On the Text tab, open the Label node.

  4. Tick Display. 

  5. By default, the label will display to the left or the corresponding curve. Tick Alternate Label Position to change the label position to the right of the curve. 

  6. Select Font to change the font attributes of the label.

  7. Select a Label Arrow.

  8. Select the Label Type, either Name of Mnemonic.

  9. Select the Label Text Background. If opaque is selected, you can then specify the Label Text Background Color.

  10. Enter the distance from the top of the plot you wish to see the first label appear in the First Label From Top of Grid.

  11. Enter the distance between labels in Repeat Labels Every box.

  12. Select Apply to implement the changes.
