Change Curve Attributes using Context Tab

  1. Select the curve on the plot. The selected curve will retain its color and all other curves will become grey. The Settings context tab will appear.

  2. Change the curve Name, Mnemonic, Units, Track, Scale, Pen and Legend. 

  3. Hit Enter to implement changes.
