Presenting TVD

You can present the True Vertical Depth (TVD) in your log, just by itself, or together with the Measured Depth (MD).

 

In GEO, we call those two depth references as primary and alternate depth system.

 

Since we do not build any algorithm to convert the MD into TVD in the software, the user needs to have the MD versus TVD data loaded as a curve. Such data can often be obtained from the vendor of survey data (nowadays, often the MWD company). Then the software needs to be told which data is going to be used for MD-TVD calculation and determine which depth system is going to be displayed.

 

Inform the software which data is going to be used for MD-TVD calculation

First, load the MD (Measured Depth) versus TVD (True Vertical Depth) curve data. Click here for instructions on loading curve data.

 

  1. On the Depth tab in the Conversion group, select TVD from the Curve drop list.

  2. Check the Use Data for TVD Conversion box.

  3. If a subsea correction needs to be applied, check TVDSS Correction and enter the value in the TVDSS Correction text box. To present it as a negative value, tick TVDSS as Negative.

 

Which depth system will display on the Plot?

  1. On the Edit tab, click Track Settings to open the Track Settings dialog box.
  2. Open the Legend and Index tab.
  3. Select the Depth track.
  4. In the Primary and Alternative Index System Settings, open the Display list and choose the depth system you wish to present. 
    Note. Please remember that the Primary and alternative depth system refer to the MD and TVD. Either MD or TVD can be the Primary or the alternative - it is defined in global settings and expanded on below.
  5. The position of the primary depth system can be justified by selecting Centre, Left or Right justify from the Primary Justification box
  6. Depth ticks can be placed for primary and/or alternate, by selecting the corresponding depths in the Ticks drop list. 
  7. The font type, style, size and color of the depth value of each depth system legend, can be selected by clicking Font in the Primary and Alternate Index Settings. 
  8. Click OK to close the Track Settings dialog box.

 

To specify which depths are primary or alternative, click Global Settings on the Edit tab. The Global Settings dialog box will open. 

  • To define MD as the primary depth, select the MD button in the  Primary Depth System section. TVD will be automatically selected as the Alternate Depth System. In this case, if Primary & Alternate is selected at the Display box of the Track Settings dialog box, both MD and TVD will be displayed.

  • To compress the whole log into TVD select TVD as your Primary Depth System in Global Settings. Then, in the Track Settings open the Legends and Depths tab anbd select Primary Only in the Display list.
