View Curve Data History

View the load and amendment history of curve data.

 

  1. On the Curves tab, select History. The View Curve Data History dialog box will open.

  2. In Display History for, select the curve you wish to view the history of. 

  3. This curve history will be detailed below. 

    Column
    The number of columns of data associated with the file loaded. 

    Start Depth
    The start depth of the data loaded. 

    End Depth
    The end depth of the data loaded. 

    File ID
    The File Name defined in the Load Data Wizard. If a curve is modified in any way, the type of modification will be detailed here (e.g [Edited], [Spliced] etc.)
    For more information on curve types, click here. 

    Date Time
    The date and time of the file load or curve amendment event. 

    User
    The logged-on user, responsible for the file load or curve amendment event. 

    Vendor
    The supplied of the curve data.

    Type
    The type of data, such as LAS, TXT, ASCII.

    File Name
    The ‘Host’ or ‘Machine ID’, file path, file name (ID) and the license number.  

    Note. You can resize this dialog box, as well as the widths of the data columns. The next time this dialog box is opened, it will use the last-sized widths.
