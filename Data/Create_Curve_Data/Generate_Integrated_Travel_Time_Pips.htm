Generate Integrated Travel Time Pips

Display integrated travel time data as pips on the log plot. This is a useful tool for integrating seismic time with depth. 

  1. Load the appropriate integrated travel time data into your ODF via the load wizard.

  2. On the Define menu, select Pip Curve Definition. The Pip Curve Definition dialog box will open. 

  3. In Select a Pip Definition, select <Create New>.

  4. Enter an appropriate Name. 

  5. Select the Source Curve for Creating Pip Data.

  6. If the source curve data is measured per foot, select the Source Curve Data Values per Feet. If the source curve data is measured per meter, select the Source Curve Data Values per Meter.

  7. In Create Minor Pip Every, enter the time interval, in microseconds (us), that the minor pips should display at. For typical integrated travel time displays, a minor pip would display every 1 millisecond (ms).

  8. In Create Major Pip Every, enter the time interval, in microseconds (us), that the major pips should display at. For typical integrated travel time displays, a major pip would display every 10 milliseconds (ms).

  9. In Data Value for Minor Pips, enter a value for the major pip display length. This will be influenced by the scale defined when displaying the pips.

  10. In Data Value for Major Pips, enter a value for the major pip display length. This will be influenced by the scale defined when displaying the pips.

  11. Select the Destination Curve for Pip Data. 
    To create a new pip curve, select an empty number and enter its mnemonic in the Mnemonic for New Pip Curve. 
    If you wish the pip curve to occupy a specific mnemonic entry, select <Curve from Mnemonic> and then define select the mnemonic from the adjacent box.

  12. Click Update. 

The pips can now be displayed on the plot.
