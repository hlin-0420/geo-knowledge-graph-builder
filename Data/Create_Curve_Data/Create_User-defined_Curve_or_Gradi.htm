Create User-defined Curve or Gradient

  1. On the Curves tab, select Curve Data. The Curve Data dialog box will open.

  2. From Select a Curve, select an [Unused] curve number. 

  3. Enter a Mnemonic.

  4. Select Create Curve. 
    The start and end depths for the database will appear in the Depth and Value columns of the two rows created.

  5. Tick Allow Editing. 

  6. Edit the Depth and Value as required.

  7. To add more depth and value data, select Append Row. 

  8. To delete rows, select the row and hit Delete on your keyboard. 
    Note. All rows displayed in the grid will be used as curve points. So, empty rows (no data) will be treated as valid data points. 

  9. Select Exit to close.

 

Tip. Over a wide depth range (more than 500ft) you should always define the curve or gradient with at least one intermediate data point (that is, use a minimum of 3 data points to define the curve or gradient).

The display settings for the new curve are defined in the Curve Settings sidebar . This can be opened by selecting Curves.   ↵
