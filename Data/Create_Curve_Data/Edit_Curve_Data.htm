Edit Curve Data

  1. On the Curves tab, select Curve Data. The Curve Data dialog box will open.

  2. From Select a Curve, select the curve you wish to edit. 

  3. Tick Allow Editing. 

  4. At the appropriate depth, select the value cell you wish to edit. 

  5. Type the new value and select the Enter key. 
    Note. Edited values appear in red, to distinguish them from the original data values in black. 

  6. Select Exit to end the editing process and close Curve Data. 

 

Tips. After editing the data, the original values are also stored in the Curve Data dialog box, under the Original Value column. You will see this column if you re-open Curve Data after editing the data. You can toggle between edited and original values by selecting either Edited Data or Original Data in the For this Curve Use list.
