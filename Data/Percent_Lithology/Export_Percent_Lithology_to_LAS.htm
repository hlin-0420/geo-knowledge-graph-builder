Export Percent Lithology to LAS

  1. On the Export tab, select Export Curve(s). The Export Curve Data dialog box will open.

  2. In Export Format, select LAS File.

  3. Check Export %-Lithology.

  4. From the Select %-Lithology Track, select the track containig the percent lithology you wish to export to LAS.

  5. Change the Start Depth, End Depth and Null Value if necessary.

  6. Choose the appropriate value for the Parameter Information from and the Well Information from.

  7. Click Export, choose a folder and file-name to export to and click Save.
