Delete Lithology

Delete lithology using the right click menu or the Definition Tool tab. 

Using the right click menu

  1. Right click on the lithology unit.

  2. From the menu, select Delete.

  3. GEO Message: 2160 will invoke – click Yes.

 

Using the Definition Tool tab

  1. Select the lithology unit. 
    This will remain in color and unselected units will become monochrome. 
    The Definition Tool tab will appear.

  2. Select the Delete button.
