Starting a new GEO session

GEO is launched using any of the following methods;

  1. Click the program icon (GEO for Windows) in the Windows menu.
    Note. The GEO icon may not appear in the Start menu anymore, when you do a fresh installation of this current version of GEO.

  2. Click the Windows button,  select Geologix GEO Suite 8, then GEO.

  3. In Windows Explorer, locate the installation location and double click the Geo.exe file.
