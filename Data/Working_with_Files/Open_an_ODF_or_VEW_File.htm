Open an ODF or VEW File

  1. On the GEO tab select Open.
    There are several ways to locate the file you wish to open.

    1. Select Recent to view all recently opened .odf and .vew files in the right panel, then select the one you wish to open.

    2. Select your OneDrive or Sharepoint from the Open panel - the files and folders will be listed on the right. Select the desired .odf or .vew file you wish to open. 
      Note. If Sharepoint or OneDrive are not listed, you may need Sign In first. 

    3. Click Browse to locate the file on your computer.
      In the Open dialog box select the location of the file you wish to open, select the file and Open. 
      Note. Ensure the relevant files type is selected in the Open dialog box.
