View File (VEW)

A VEW file contains information related to data presentation, such as the track layout and log scale. 


It does not contain data.


A VEW file can be loaded into a blank GEO session to create an initial layout guide, or into an ODF to present existing data in a different format - for example, a litholog can be converted into a complog, simply by loading a VEW file which has been formatted to present in a complog style.
